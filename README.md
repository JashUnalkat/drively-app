# 🚗 Drively Mobile Application

---

## 📖 Project Overview

Drively is a cross-platform mobile application developed using **React Native** and **Expo** to help users improve their driving knowledge, develop safer driving habits, and manage essential vehicle information in one convenient platform.

The application combines educational resources, vehicle maintenance tracking, insurance services, and interactive practice quizzes to provide users with a comprehensive driving companion. It is designed with a modern, responsive interface that delivers a smooth, intuitive, and engaging user experience across mobile devices.

Built using a component-based architecture, Drively demonstrates practical implementation of mobile application development, Firebase authentication, cloud data management, and responsive user interface design.

---

## ⭐ Project Highlights

- Cross-platform mobile application built with React Native and Expo
- Secure user authentication using Firebase Authentication
- Cloud-based data management with Firebase Firestore
- Responsive and user-friendly interface
- Interactive learning resources and practice quizzes
- Vehicle maintenance tracking and insurance management
- Modular architecture for scalability and maintainability

---

## 🚀 Features

### 🔐 Authentication

- Secure user registration and login using Firebase Authentication
- User account creation with:
  - Full Name
  - Email Address
  - Password
- Secure authentication and session management

---

### 🏠 Dashboard

After successful authentication, users are redirected to a personalized dashboard where they can:

- View their driving performance score
- Monitor driving safety level using a visual gauge
- Access all application features from a centralized interface

---

### 👤 Profile Management

- View personal profile information
- Update user details
- Logout securely
- Access FAQ and Privacy Policy pages

---

### 🚗 Driving Behaviors

- Monitor positive and negative driving habits
- Improve driving awareness through personalized feedback
- Track overall driving performance

---

### 🎓 Learning & Practice

#### 📹 Educational Videos

Users can access educational videos covering topics such as:

- Winter Driving
- Rain Driving
- Defensive Driving
- Road Safety Precautions

#### 📝 Practice MCQs

Practice tests designed to prepare users for Ontario driving examinations.

Available tests include:

- G1 Practice Test
- G2 Practice Test
- G Road Test

Features:

- 15 questions per test
- Instant score calculation
- Review incorrect answers
- Learn from correct answers after each attempt

---

### 🔧 Vehicle Maintenance

Track important vehicle maintenance activities including:

- Oil Change Reminders
- Tire Rotation Schedule
- Brake Inspection Tracking
- Battery Health Monitoring

---

### 🛡️ Insurance Services

Users can generate insurance quotes by entering:

- Driver License Type
- Years of Driving Experience
- Vehicle Information

Supported providers include:

- Aviva
- TD Insurance
- Intact Insurance

Additional Features:

- Compare insurance plans
- Select preferred coverage
- Complete payment process
- Receive payment confirmation

## 🚀 Installation

Clone the repository:

```bash

```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npx expo start
```

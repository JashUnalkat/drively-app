
# Drively Mobile Application 🚗  


---

## 📌 Project Overview
Drively is a mobile application developed using React Native on Expo Snack. This application is designed to help users improve their driving habits, enhance road safety awareness, and access useful tools related to driving.

The application was developed and improved by **Dinh Huy Nguyen** and **Jash Unalkat** as part of the final project for the INFO-5139 course.

The main focus of this project is on user interface design, smooth navigation, and providing a complete user experience through multiple integrated features.

---

## 🚀 Features

### 🔐 Authentication
- Users can log in using their email and password
- New users can create an account by entering:
  - Full Name
  - Email
  - Password
- After registration, users can log in using their credentials

---

### 🏠 Dashboard
After successful login, users are directed to the dashboard:
- Displays driving score
- Shows safety level using a visual gauge
- Provides quick access to all main features of the app

---

### 👤 Profile Section
- Displays user information such as name and email
- Option to edit personal details
- Logout functionality
- Includes support options like FAQ and Privacy Policy

---

### 🚗 Behaviors
- Displays driving habits of the user
- Includes both positive and negative behaviors
- Helps users understand and improve their driving performance

---

### 🎓 Scenarios

This section is designed for learning and practice.

#### 📹 Video Learning
- Users can watch educational driving videos
- Topics include:
  - Winter driving
  - Rain driving
  - Safety precautions

#### 📝 Practice MCQ
- Includes three test types:
  - G1 Test
  - G2 Test
  - G Test
- Each test contains 15 questions
- After completion:
  - User receives score
  - Incorrect answers are shown
  - Correct answers are provided for learning

---

### 🔧 Maintenance
- Helps users track vehicle maintenance
- Includes:
  - Oil change reminders
  - Tire rotation schedule
  - Brake check updates
  - Battery health tracking

---

### 🛡️ Insurance
- Users can generate insurance quotes
- Input required details:
  - License type
  - Years of driving
  - Vehicle details
- Compare providers such as:
  - Aviva
  - TD Insurance
  - Intact Insurance
- Users can select a plan and complete payment
- Payment confirmation screen is displayed after successful transaction

---

## 🛠️ Technologies Used
- React Native
- Expo Snack
- JavaScript
- React Navigation



---

## 📱 Application Flow

1. User opens the application  
2. Login screen appears  
3. New users can sign up  
4. Existing users log in using credentials  
5. Dashboard is displayed  
6. User navigates through:
   - Behaviors
   - Scenarios
   - Maintenance
   - Insurance  
7. Profile section allows editing details or logging out
